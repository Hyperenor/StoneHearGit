UC01 : Créer un compte 
16/10/2017 - Version 1.1 - Lachachi Charaf
Description : L'utilisateur souhaite créer un compte .
Acteur : Utilisateur
Précondition : /
Scenario nominal :
1.	L'utilisateur choisit "créer un compte".
2.	Le système affiche le formulaire d'ajout.
3.	L'utilisateur introduit ses informations et valide.
4.	Le système vérifie si les informations requises ont bien été introduites.
5.	Le système enregistre les informations.
Post conditions :
	Mise à jour du nombre d'utilisateur inscrit.
Alternative :
1.	Les informations requises n'ont pas été introduites
a.	Le système affiche une alerte « veuillez renseigner tous les champs obligatoires »
b.	Retour en SN2
Exceptions :
1.	Annulation
a.	En SN3 ou SN5 l'utilisateur peut choisir d'annuler 
b.	Le système ferme le formulaire d'ajout.
